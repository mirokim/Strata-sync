- `chief persona.md` — 허브 문서
- 최신: `[2026_02_27] 프로젝트A 이사장님 피드백.md`

### 수정 전 문제 (chief_director 페르소나)

### 수정 후 개선
- **Fix #7**: TAG_BOOST 곱셈 전환 → base 0.15 × 1.1 = 0.165 (가산 0.25 대비 낮음)
- base score가 높은 최신 피드백만 TAG_BOOST 효과가 유의미
- `RECENCY_INTENT_RE` "최근" 매칭 → 날짜순 재정렬 → 2026-02-27 피드백 우선
