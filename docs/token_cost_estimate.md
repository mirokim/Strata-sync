### Haiku 전환 현황 (2026-03-25 적용)

아래 함수들은 단순 분류/요약/변환 작업이므로 메인 모델(Sonnet) 대신 **워커 모델(Haiku)**로 전환하여 비용을 73% 절감.

| 함수 | 파일 | 작업 내용 | 변경 전 | 변경 후 |

| `suggestSpeakerForDoc` | tagService.ts | 화자(페르소나) 5지선다 분류 | Sonnet | **Haiku** |

## 2. 시나리오별 비용 추정

### ① 단순 채팅 1회 (1 페르소나)

| 구성 요소 | Input 토큰 | Output 토큰 |
|-----------|-----------|------------|
| 시스템 프롬프트 (페르소나 바이오 + 지침) | ~1,500 | — |
| RAG 컨텍스트 (BM25+TF-IDF 결과 주입, 최대 20K chars) | ~5,000 | — |
| 메시지 히스토리 (최근 ~10개, 최대 20K chars) | ~5,000 | — |
| 사용자 입력 + 첨부 | ~500 | — |
| 응답 생성 | — | ~1,000 |
| **합계** | **~12,000** | **~1,000** |

**비용**: (12K × $3 + 1K × $15) / 1M = **$0.051** (~67원)

> 참고: BM25/TF-IDF 검색 자체는 로컬 연산이므로 토큰 비용 $0.
> RAG worker 발동(문서 요약) 시 Haiku 추가 호출 → +$0.01~0.02

### ② 멀티 페르소나 동시 채팅 (N명)

각 페르소나마다 독립된 LLM 호출. RAG 컨텍스트는 동일하지만 시스템 프롬프트(바이오)가 다름.

| 페르소나 수 | LLM 호출 | 추정 비용 (Sonnet) | 원화 |
|------------|---------|-------------------|------|
| 1명 | 1 | $0.05 | ~67원 |
| 2명 | 2 | $0.10 | ~135원 |
| **3명** | **3** | **$0.15** | **~200원** |
| 4명 | 4 | $0.20 | ~270원 |
| 5명 | 5 | $0.26 | ~340원 |

---

### ③ RAG Worker 발동 (깊은 검색)

BM25 hit가 강할 때(score ≥ 0.4, 3개+ 매칭) worker LLM이 문서 요약을 병렬 수행.

| 구성 요소 | 모델 | Input | Output | 비용 |
|-----------|------|-------|--------|------|
| Worker 요약 × 5문서 | Haiku | 5 × 2K = 10K | 5 × 200 = 1K | $0.012 |
| Domain 합성 × 2도메인 | Sonnet | 2 × 1.2K = 2.4K | 2 × 300 = 600 | $0.016 |
| 메인 응답 (RAG 컨텍스트 확장) | Sonnet | ~18K | ~1K | $0.069 |
| **합계** | | | | **$0.097** (~130원) |

> 단순 채팅 대비 +$0.04 (약 1.9배)

### ⑨ 태그/화자 일괄 분류 (Bulk Assign)

전체 볼트 문서에 AI 태그·화자 자동 분류. **Haiku 사용** (v2 전환).

| 항목 | 값 |
|------|------|
| 문서 수 | ~1,200 |
| 호출 수 | 1,200 (태그) + 1,200 (화자) = 2,400 |
| Input/호출 | ~400 토큰 (300자 프리뷰 + 프롬프트) |
| Output/호출 | ~30 토큰 (JSON 배열 또는 ID 1개) |
| 모델 | Haiku ($0.80/$4.00) |
| **총 비용** | 2,400 × (400×0.8 + 30×4) / 1M = **$1.06** (~1,400원) |

> Sonnet 기준이었으면 $3.84 (~5,100원). Haiku 전환으로 **-72%**.

## 5. 비용 모니터링

Sandbox Map은 모든 LLM 호출을 `usageStore`에 자동 기록합니다.

- **Settings → Usage 탭**: 세션 누적 토큰/비용 확인
- **호출별 로그**: 최근 500건 (localStorage)
- **caller 라벨**: `chat`, `editAgent`, `debate`, `report`, `graphInsight`
- **리셋**: Settings에서 수동 초기화

> 코드: `src/stores/usageStore.ts`
